\subsection{Detailed Description}
Multicast discovery stage. 


\begin{DoxyItemize}
\item Request

The end host sends an \hyperlink{classndn_1_1Interest}{Interest} over a multicast face.

\hyperlink{classndn_1_1Interest}{Interest} \hyperlink{classndn_1_1Name}{Name} is /localhop/ndn-\/autoconf/hub.
\item Response

A producer app on the H\+UB answer this \hyperlink{classndn_1_1Interest}{Interest} with a \hyperlink{classndn_1_1Data}{Data} packet that contains a T\+L\+V-\/encoded Uri block. The value of this block is the U\+RI for the H\+UB, preferably a U\+DP tunnel. 
\end{DoxyItemize}
