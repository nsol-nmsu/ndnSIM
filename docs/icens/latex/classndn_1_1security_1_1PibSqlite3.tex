\subsection{Detailed Description}
\hyperlink{classndn_1_1security_1_1Pib}{Pib} backend implementation based on S\+Q\+Lite3 database. 

All the contents in \hyperlink{classndn_1_1security_1_1Pib}{Pib} are stored in a S\+Q\+Lite3 database file. This backend provides more persistent storage than \hyperlink{classndn_1_1security_1_1PibMemory}{Pib\+Memory}.
