\subsection{Detailed Description}
Executes N\+FD with R\+IB manager. 

N\+FD (main forwarding procedure) and R\+IB manager execute in two different threads. Each thread has its own instances global io\+\_\+service and global scheduler.

When either of the daemons fails, execution of non-\/failed daemon will be terminated as well. In other words, when N\+FD fails, R\+IB manager will be terminated; when R\+IB manager fails, N\+FD will be terminated.
