\subsection{Detailed Description}
Implementation network-\/layer of N\+DN stack. 

This class defines the A\+PI to manipulate the following aspects of the N\+DN stack implementation\+:
\begin{DoxyEnumerate}
\item register a face (Face-\/derived object) for use by the N\+DN layer
\end{DoxyEnumerate}

Each Face-\/derived object has conceptually a single N\+DN interface associated with it.

In addition, this class defines N\+DN packet coding constants
