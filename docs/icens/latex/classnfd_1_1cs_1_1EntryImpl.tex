\subsection{Detailed Description}
an \hyperlink{classnfd_1_1cs_1_1Entry}{Entry} in Content\+Store implementation 

An \hyperlink{classnfd_1_1cs_1_1Entry}{Entry} is either a stored \hyperlink{classnfd_1_1cs_1_1Entry}{Entry} which contains a Data packet and related attributes, or a query \hyperlink{classnfd_1_1cs_1_1Entry}{Entry} which contains a Name that is Less\+Comparable to other stored/query \hyperlink{classnfd_1_1cs_1_1Entry}{Entry} and is used to lookup a container of entries.

\begin{DoxyNote}{Note}
This type is internal to this specific Content\+Store implementation. 
\end{DoxyNote}
