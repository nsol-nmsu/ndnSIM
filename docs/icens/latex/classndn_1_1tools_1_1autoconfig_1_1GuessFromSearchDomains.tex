\subsection{Detailed Description}
Guessing home router based on D\+NS query with default suffix. 


\begin{DoxyItemize}
\item Request

The end host sends a D\+NS query that is equivalent to this command\+: \begin{DoxyVerb}dig +search +short +cmd +tries=2 +ndots=10 _ndn._udp srv
\end{DoxyVerb}

\item Response

The D\+NS server should answer with an S\+RV record that contains the hostname and U\+DP port number of the N\+DN router. 
\end{DoxyItemize}
