\subsection{Detailed Description}
Forwarding Information Base (F\+IB) helper. 

The F\+IB helper interacts with the F\+IB manager of N\+FD by sending special Interest commands to the manager in order to add/remove a next hop from F\+IB entries or add routes to the F\+IB manually (manual configuration of F\+IB).
