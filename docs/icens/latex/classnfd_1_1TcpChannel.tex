\subsection{Detailed Description}
Class implementing T\+C\+P-\/based channel to create faces. 

\hyperlink{classnfd_1_1Channel}{Channel} can create faces as a response to incoming T\+CP connections (\hyperlink{classnfd_1_1TcpChannel_ab2c06032e8775433fc01ce11937f7ca9}{Tcp\+Channel\+::listen} needs to be called for that to work) or explicitly after using \hyperlink{classnfd_1_1TcpChannel_aecd445a7df92fd55daa14bb6059796f4}{Tcp\+Channel\+::connect} method.
