\subsection{Detailed Description}
Class representing a general-\/use automatically managed/resized buffer. 

In most respect, \hyperlink{classndn_1_1Buffer}{Buffer} class is equivalent to std\+::vector$<$uint8\+\_\+t$>$ and is in fact uses it as a base class. In addition to that, it provides \hyperlink{classndn_1_1Buffer_a1ab623bc5bc99475676e42ff7b41a0f4}{buf()} and \hyperlink{classndn_1_1Buffer_a1ab623bc5bc99475676e42ff7b41a0f4}{buf$<$\+T$>$()} helper method for easier access to the underlying data (\hyperlink{classndn_1_1Buffer_a1ab623bc5bc99475676e42ff7b41a0f4}{buf$<$\+T$>$()} casts pointer to the requested class)
